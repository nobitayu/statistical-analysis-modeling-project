# 快速启动指南

## 1. 安装依赖

```bash
pip install -r requirements_system.txt
```

或者使用项目的requirements.txt：

```bash
pip install streamlit
```

## 2. 运行系统

### Windows系统
```bash
run.bat
```

或者直接运行：
```bash
streamlit run app.py
```

### Linux/Mac系统
```bash
chmod +x run.sh
./run.sh
```

或者直接运行：
```bash
streamlit run app.py
```

## 3. 访问系统

系统启动后，浏览器会自动打开，地址为：
```
http://localhost:8501
```

如果没有自动打开，请手动在浏览器中输入上述地址。
